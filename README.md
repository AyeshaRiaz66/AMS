# 📊 Academic Evaluation System

A console-based academic result management system developed in C++ using structured and object-oriented programming.

🚀 Features
- Input student assessment components (quizzes, assignments, midterm, final)
- Automatic calculation of total marks and percentage
- GPA/average computation
- Letter grade assignment based on grading criteria
- Performance remarks generation (Excellent, Good, Needs Improvement)

🛠️ Technologies Used
- C++
- OOP Principles
- Conditional Logic

📚 Concepts Applied
- Structured Programming
- Object-Oriented Programming
- Control Statements
- Algorithmic Calculations

 🎯 Objective
To automate academic performance evaluation and grading logic through efficient program design.
